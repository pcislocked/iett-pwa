# Holistic Architectural Review Tasks

- [ ] **Accessibility (a11y) & UX**: In `MapPage.tsx`, the `showErrorModal` handles `Escape` globally but fails to restore focus to the previously active element upon closing, leaving screen reader and keyboard users without context. (Compare with `BusDetailSheet` in `StopPage.tsx` which handles this correctly).
- [ ] **Silent Failure & Missing UI**: In `StopPage.tsx`, the `showAnnouncements` state is defined and `announcements` are fetched via `annsFetcher`, but they are **never rendered** in the JSX. This is a complete feature drop.
- [ ] **Silent Failure in Error Handling**: In `StopPage.tsx` `annsFetcher`, if partial failures occur during the Promise.allSettled chunking (e.g., 2 out of 5 requests fail), it silently drops the failed route announcements and only throws if *all* requests fail. It needs to expose partial errors or retry.
- [ ] **Race Condition**: In `MapPage.tsx` `fetchBuses`, if a user selects, deselects, and quickly re-selects a route, `inFlight.current.has(route)` prevents a new fetch. When the original fetch resolves, `selectedRoutesRef.current.includes(route)` is true, meaning stale data from the original fetch is applied to the new selection state.
- [ ] **State Update on Unmounted Component**: In `MapPage.tsx` `fetchBuses`, there is no `isMounted` ref check. If the component unmounts while custom polled bus requests are in flight, the `.then` block will call `setRouteBusMap`, triggering a React state update warning / potential memory leak.
- [ ] **Resource Leak**: In `StopPage.tsx` `MapResizer`, `requestAnimationFrame` is used to call `map.invalidateSize()` but the frame ID is not captured and cancelled on unmount. This will cause an exception if the component unmounts before the frame fires.
- [ ] **Test Hermeticity**: Add specific tests for `StopPage` `annsFetcher` to simulate partial HTTP failures and verify that the UI handles them correctly instead of swallowing them.
- [ ] **Test Hermeticity**: Add a test in `MapPage` testing the rapid select/deselect/re-select race condition to ensure stale promises are properly invalidated using AbortController or a request ID system.
